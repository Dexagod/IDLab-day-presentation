Integrating personal data in the Web ecosystem

Ruben Dedecker

We need integration of our data
in the decentralized Web ecosystem

We can govern the integration of data
on the powers that govern society

  • legislative power - controling data
  • executive power - sharing data
  • judiciary power - detecting misuse

Integration of our data enables
dialog between agents and services

What is our focus? Dialog? Reasoning? ??? We are missing a thread somewhere

Entities define rules and preferences

This represent legislative power

Data spaces expose flexible interfaces

This represent executive power

Agents detect misuse through receipts

This represent judicial power

Integration of personal data in the Web
by fulfilling the governing functions

  • executive power
  • legislative power
  • judicial power

dialog

Conclusion

  • With dynamic governance of our data on the Web
  • we enable automated dialog between agents
  • over all aspects of our data
